URL Shortener Microservice

To Use This Microservice

Add /new/URL-TO-SHORTEN to the current URL, and you will be provided with a shortened URL.

Note, do not include the protocol (e.g. http:// or https://).

For example:

https://nai888-fcc-ls.glitch.me/new/google.com

https://nai888-fcc-ls.glitch.me/0
