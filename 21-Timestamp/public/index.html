Timestamp Microservice

To Use This Microservice

Navigate to the current URL with a date added to the end.

For example:

https://nai888-fcc-timestamp-microservice.glitch.me/July%2015,%202017

https://nai888-fcc-timestamp-microservice.glitch.me/1500094800
