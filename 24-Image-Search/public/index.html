Image Search Abstraction Layer

There are two aspects to this microservice:

1. Add /search/SEARCH-QUERY to the current URL to search for images matching your query. Paginate through the responses by adding ?offset= and a number.

For example:

https://nai888-fcc-imagesearch.glitch.me/search/lolcats?offset=2

2. Visit https://nai888-fcc-imagesearch.glitch.me/latest to view the ten most recent searches.
